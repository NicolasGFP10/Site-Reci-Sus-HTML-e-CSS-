#### O QUE É O RECI SUS

<img width="1366" height="739" alt="imagem do reci sus" src="https://github.com/user-attachments/assets/03cb6bbb-bb25-4555-9908-a70902f2549c" />

O RECI SUS é um site criado para a concientização da reciclagem sustentável, criado no meu período de estudo na ETEC de Nova Odessa

Os impactos gerados pelo descarte incorreto prejudicam muito além de rios e mares, considerados os mais atingidos. Nossos solos ficam extremamente propensos a riscos como erosão, contaminação, desequilíbrio e perda de nutrientes.

Não é novidade que a poluição pelo descarte incorreto de lixo pode gerar problemas irreparáveis para a biodiversidade do planeta. Um dos principais fatores que culminam nesses problemas é a intoxicação por plástico, produtos tóxicos e metais pesados pela fauna e até mesmo pelas plantas.
